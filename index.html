Дорогие гости музея!

Рады приветствовать вас в стенах нашего музея!

Ваш увлекательный путь по экспозициям начнётся с особой системы навигации. По всему музею вы найдёте пронумерованные указатели, которые помогут вам ориентироваться в экспозициях.

Для комфортного знакомства с выставкой рекомендуем следовать нумерации:

    Начните своё путешествие по музею с вводной части под номером 1, где собраны основные материалы экспозиции

    Далее двигайтесь по возрастанию номеров

    Вы можете воспользоваться аудиогидом через мобильное устройство (телефон или планшет).

    Просим вас соблюдать тишину во время прослушивания аудиогида, чтобы не нарушать атмосферу музея для других посетителей. Благодарим за понимание!

    Чтобы начать ваше путешествие по музею, нажмите на кнопку ниже.

    Начать осмотр (1)
